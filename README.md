# Chrome_App
vanilla js로 만든 오늘을 기록하는 웹사이트

# 링크
https://skgml0.github.io/Chrome_App/

# 구현 완료
- 현재 주소로 날씨 정보 알려주기

- Local Storage 이용하여 To Do 저장, 이름 저장 

- 커피 API를 이용하여 랜덤으로 오늘의 커피를 추천

- 저장한 명언 중 랜덤으로 하나의 랜덤 보여주기 

- [사이트?] 를 클릭하면 모달창 띄우기 (모달창 바깥 클릭 시 닫힘)
  

# 구현 중
- 슬라이드 앨범 or 슬라이드 페이지

- 메모없을때 남는 범위 버그 없애기

- 음악 넣기

- todo 중 기록할 것 따로 남기기
